# **App Name**: BeatBuddy Metronome

## Core Features:

- Start/Stop Control: A prominent, easy-to-tap button to start and stop the metronome beat.
- BPM Slider Adjustment: An intuitive slider for quickly and smoothly adjusting the beats per minute (BPM).
- Customizable Time Signatures: Ability to set various time signatures (e.g., 4/4, 3/4) to suit different musical pieces.
- Visual Beat Indicator: A clear, dynamic visual indicator that illuminates or changes with each beat for easy tracking.
- Practice Prompt Generator: An AI-powered tool that generates simple rhythm exercises or practice prompts based on the current BPM and time signature, suitable for young learners.

## Style Guidelines:

- Overall scheme: Sleek dark mode to reduce eye strain, creating a focused environment for learning.
- Primary vibrant accent: #6BEF2B (a vivid green) for interactive elements like the start/stop button and beat indicator, symbolizing growth and energy.
- Background dark tone: #1D2518 (a very dark muted green) providing a subtle depth that complements the vibrant primary accent.
- Secondary accent: #A2EC93 (a light yellow-green) used for minor UI elements, providing additional visual hierarchy and warmth.
- Font choice: 'PT Sans' (humanist sans-serif) for all text, ensuring high readability and a friendly, modern feel suitable for children.
- Icons: Use large, simple, and universally recognizable icons that are easy for young children to understand and tap.
- Layout: Uncluttered interface with prominent, centralized interactive elements. Ample spacing to ensure large, easy-to-target touch areas for small hands.
- Animations: Smooth transitions and clear visual feedback for button presses, slider adjustments, and especially for the beat indicator to enhance the user experience and clarity.
